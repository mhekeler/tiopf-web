tiOPF Support by news groups (English)

NNTP (news) server setup:

News Server: opensoft.homeip.net
News Server port: 119
   
Web Interface: http://opensoft.homeip.net:8080/webnews/
User name: anything as long as it is a single alphanumeric name.

Currently we have 4 newsgroups setup:

tiOPF.development: This is where the people developing the framework can communicate about future or current issues.

tiOPF.support: Anybody only using the framework and not helping to develop it, needs to post your questions and answers in here.

tiOPF.binary: When it is difficult to explain a problem, a small sample application might help. This group allows for small attachments in .zip format.

tiOPF.dailybuilds: The daily unit tests status emails get sent to this area. No posting is allowed in here.
